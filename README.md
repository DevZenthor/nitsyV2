# 🎨 Nitsy Portfolio

Site portfolio moderne et responsive créé pour **Nitsy**, graphiste spécialisé en :

* 🖼️ Miniatures YouTube
* 🎯 Logos
* 📌 Posters / Affiches
* ✨ Créations visuelles sur mesure

Développé avec **React + Vite** avec une direction artistique moderne et naturelle 🌿

---

## 🚀 Aperçu

Le site contient :

* 🏠 Home page premium
* 🎨 Portfolio filtrable
* 🤝 Section Clients
* 💼 Services avec tarifs
* ⚖️ Mentions légales
* 📱 Responsive mobile complet
* 🍔 Navbar burger animée
* 🌿 UI moderne avec effets lumineux

---

## 🛠️ Technologies utilisées

* React.js
* Vite
* React Router DOM
* React Icons
* CSS3
* Bootstrap Grid

## 📦 Installation

Clone le projet :

```bash
git clone https://github.com/tonpseudo/nitsy-portfolio.git
```

Entre dans le dossier :

```bash
cd nitsy-portfolio
```

Installe les dépendances :

```bash
npm install
```

Lance le serveur de développement :

```bash
npm run dev
```
